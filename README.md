# Credit-Card-Transaction-Dashboard
This repository contains a complete end-to-end Credit Card Financial &amp; Customer Analytics project designed to analyze customer behavior, spending patterns, revenue trends, and key performance metrics for a financial institution.
Developed a complete Credit Card Financial & Customer Analytics project using MySQL and Power BI to analyze revenue, customer behavior, and transaction patterns.
Calculated major KPIs: Total Revenue, Total Transactions, Average Spend, High-Value Customers.
Identified top-performing card types, categories (shopping, travel, EMI) and customer demographics.
Analyzed monthly spending trends, category distribution, and gender/age-based revenue patterns.
